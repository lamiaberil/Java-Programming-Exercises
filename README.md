# Java-Programming-Exercises
Bu depo, eğitim sürecimde geliştirdiğim temel ve orta seviye Java uygulamalarını içermektedir.

Kütüphane Takip Sistemi: Custom Exception Handling ve kullanıcı veri doğrulaması üzerine odaklanılmıştır.

Öğrenci Analiz Sistemi: Java Stream API ve Lambda ifadeleri kullanılarak fonksiyonel veri işleme yapılmıştır.
